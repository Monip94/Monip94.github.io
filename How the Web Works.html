The World Wide Web

We as Humans are very dependant on using the web for various reasons. When we use the web we really dont see what is really happening for us to be able to use this technology. When we type into a search bar the computer automatically translates the text into binary coding. A site is then requested in items called packets from the client server that will respond with the web page if it there. If it is not there it will transfer the request until the domain is found. Once the conversation is complete a webpage will be displayed for you.
